## 🌟 Description

**HELPEthiopia** is a **full-stack donation platform** built for **Help Ethiopia Voluntary Organization**, a registered non-profit dedicated to transforming healthcare access in Ethiopia. The organization operates at **Tikur Anbessa Specialised Hospital** (Black Lion Hospital), supporting economically disadvantaged patients who cannot afford medical treatment.

You can explore patient stories, learn about the organization's impact, and make secure donations—all through a **clean, responsive, and mobile-friendly interface** powered by React and Tailwind CSS.

🔗 **Live Demo:** [https://app.helpethiopia.org.et/](https://app.helpethiopia.org.et/)

---

## 🚀 Features

- 💳 **Secure Donations:** Integrated with **Chapa payment gateway** for safe online transactions
- 📱 **Fully Responsive:** Optimized for mobile, tablet, and desktop users
- 📖 **Patient Stories:** Real-life stories of patients helped by the organization
- 📊 **Impact Metrics:** Live counters showing patients helped, volunteers, and programs
- 👥 **Team Credits:** All volunteer developers are officially credited on the website
- 🎨 **Modern UI:** Built with **React.js and Tailwind CSS** for a clean, accessible experience
- 🔐 **Admin Dashboard:** Backend interface for managing content and donations

## 🛠️ Tech Stack

### Frontend

## 🛠️ Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Ahla22-19/helpethiopia.git
   cd helpethiopia
   ```
2. **Install dependencies:**

```bash
npm install
```
3. **Set up environment variables:**
Create a .env file in the root directory:

```env
REACT_APP_API_BASE_URL=your_backend_api_url
REACT_APP_CHAPA_PUBLIC_KEY=your_chapa_public_key
```
4.**Run the development server**

```bash
npm start
```
5.**Build for production:**

```bash
npm run build
```
## 💻 Usage

Once the development server is running, open [http://localhost:3000](http://localhost:3000) to view the app in your browser.

- Browse **patient stories** on the homepage
- Click **"Donate"** to support a specific patient or the general fund
- View **impact metrics** showing how donations are helping
- Visit the **team page** to see the developers behind the project
